# 20_Hospital_Roles_Authorization System

A Role-Based Authentication System built with ASP.NET Core MVC and ASP.NET Core Identity.

This project demonstrates user authentication and authorization using different roles in a Hospital Management scenario.

---

## Features

- User Registration
- User Login
- User Logout
- ASP.NET Core Identity
- Email Validation
- Password Hashing
- Role-Based Authentication
- Automatic Role Creation
- Role Assignment During Registration
- Role-Based Login Redirection

---

## Roles

- Admin
- Doctor
- Nurse
- Receptionist
